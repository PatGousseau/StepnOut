# Short-Form Format Experiments

`render_formats.py` renders seven deliberately non-generic short-form videos. It
uses no Pexels footage: every visual is original motion graphics rendered with
Pillow and `ffmpeg`, and every narration is generated by ElevenLabs.

Run all seven:

```bash
python scripts/shortform_video/experiments/render_formats.py
```

Render one while refining it:

```bash
python scripts/shortform_video/experiments/render_formats.py --only side_quest_roulette
```

Outputs and a `manifest.json` are written to `/tmp/stepnout_shortform/experiments`.
The source deliberately keeps each visual format custom. The point is to find a
format with real viewer response before extracting a config schema from it.

## Roulette Template

The roulette experiment has now been extracted into a reusable renderer:

```bash
python scripts/shortform_video/roulette.py scripts/shortform_video/configs/roulette.json
```

Its wheel hides every candidate behind stars while it spins, then only reveals
the selected candidate's title, completion rule, and benefit. Add candidates in
`configs/roulette.json`, choose one with `roulette.winner`, or omit it and use
`--seed` to select a reproducible random outcome.

## The six hypotheses

| Format | What it tests |
| --- | --- |
| `blunt_intervention` | Confrontational reframe and kinetic typography as the hook. |
| `cancel_brain` | Comedy and recognisable internal conflict. |
| `nature_documentary` | Parody as a shareable treatment for a familiar behaviour. |
| `side_quest_roulette` | Agency, pause/replay behaviour, and comment prompts. |
| `message_thriller` | A small social risk delivered as a tension-and-release story. |
| `domino_laboratory` | A satisfying visual metaphor that pays off the instruction. |
| `comfort_zone_calendar` | A fake weekend calendar turns avoidance into an urgent, playful override. |

## What to compare

Do not decide from the rendered files alone. Publish or privately test them as
separate formats, then compare the first-second swipe rate, retention curve,
completion rate, rewatches, shares, saves, comments, and follows per view.
The winning format should be refined as a format family; the others are useful
negative controls, not failed scripts.
